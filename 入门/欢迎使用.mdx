# 欢迎使用

**53AIHub** 是一款开源的智能体（AI Agent）发布与运营平台。它实现了无缝对接字节扣子、腾讯元器、Dify、FastGPT、RAGFlow、53AI Studio等智能体开发平台以及阿里百炼、腾讯云智能体开发平台、火山方舟、百度千帆AppBuild等云计算平台，让你可以快速搭建生产运营级的AI门户。即使你是非技术人员，也能参与到智能体的发布和 AI 客户的运营中。

53AI Hub 内置了搭建和运营AI 门户所需的关键技术栈，包括主流的智能体开发平台接入的支持、完善的智能体接入和管理、稳健的底层框架、灵活的运营策略，并提供了用户运营和个性化模版布局。53AI Hub 开箱即用的产品特性为企业和 AI 门户的运营者节省了大量的开发时间，使其可以专注在业务需求和用户运营上。

### 为什么使用 53AI Hub？

你或许可以基于扣子、Dify、FastGPT、RAGFlow等智能体开发平台的分享链接将你构建的 AI应用分享给用户使用。相比之下，53AI Hub 提供了将你在不同平台开发的 AI 应用聚合在一个统一的AI门户上进行运营和管理所需要的完整解决方案，53AI Hub 是一套开箱即用的成熟系统，经过了严谨的工程设计和软件测试。

更重要的是，53AI Hub 是**开源**的，它由一个53AI的全职团队和社区共同打造。你可以选择使用53AI Hub 的云服务，也可以选择将 53AI Hub 部署在你自己的云服务器上，在灵活和安全的基础上，同时保持对用户与运营数据的完全掌控。

希望以上信息和这份指南可以帮助你了解这款产品，我们相信 53AI Hub 是您发布和运营智能体的最佳平台。

### 53AI Hub 能做什么？

53AI Hub 产品源自用户真实的业务痛点及经营需求，我们希望每个组织都能零门槛的运营<mark style="color:blue;">自己独立的AI门户</mark>。

哪些场景适合使用？

* **AI创新创业**，快速将你构建的 智能体接入到独立的AI门户进行运营 ，观测用户行为并与用户持续互动。
* **企业AI门户**，一些企业正在部署面向内部全员使用的 AI门户 ，加速生成式AI技术在企业内的推广与应用提升全员的工作效率，并实现中心化的管控。
* **AI培训老师**，一些培训机构和AI培训老师将课程中案例搭建成智能体接入到 53AIHub 供学员体验和使用 ，在 53AI Hub 的管理端是跟踪学员的学习效果及使用情况。
